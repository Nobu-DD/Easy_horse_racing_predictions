# 競馬簡単予想アプリ
## アプリ内容
競馬の馬券を購入する時に、予想をする時間がない方(自分含む)に向けた、簡単予想アプリになります。<br>
### アプリ開発の理由
私自身、プログラミング学習を始めて競馬を予想する時間が少なくなってしまい、少し競馬から離れ気味だな〜と感じていました。<br>
そこで、自分の考えた買い方を簡単にアプリにお任せする機能があればいいな〜と思い、開発に挑戦してみました。(あとXで使ってみたいというお声もお聞きしたので)
